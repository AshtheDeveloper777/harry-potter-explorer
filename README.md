## API Used

This project uses the [Harry Potter API](https://hp-api.onrender.com/) which:
- Is completely free
- Requires no API key
- Provides comprehensive data about characters, spells, and more from the Harry Potter universe

## How to Use

1. Open `index.html` in your web browser
2. Use the tabs to switch between Characters, Spells, and Houses
3. Use the search box to find specific items
4. Click on cards to view detailed information
